NYC Broken Trees v3

  • represents all 311 calls regarding damaged trees, 2010 to present (2/16/2017)
  • each orange line represents one zip code
  • each brown line represents 1 of 6 descriptions of tree damage:
    • branch cracked and will fall
    • branch or limb has fallen down
    • entire tree has fallen down
    • tree alive - in poor condition
    • tree leaning / uprooted
    • tree trunk split
  • distance of each line represents volume of calls
